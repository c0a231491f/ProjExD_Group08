# 魚

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 主人公キャラクターをキーボード操作によりオブジェクトを食べるゲームである
* 参考URL：[ドリームフィッシュ](https://wowgame.jp/game_html/21.html)

## ゲームの遊び方
* 矢印キーで主人公キャラクターを操作し，オブジェクトを食べて主人公キャラクターがでかくなる
* ライフが0になったら，ゲームオーバーとなる

## ゲームの実装
### 共通基本機能
* 背景画像（空(そら)）と主人公キャラクターの描画
* 矢印キーで主人公キャラクターを操作

### 分担追加機能
* スコア、タイム、ライフの描画（担当：）
* ライフが減ったらゲームオーバー（担当：）
* 様々な大きさのオブジェクト（担当：）
* 魚とのあたり判定（担当：）
